# Icon Layers for Liquid Glass Icon

Create these layers in Icon Composer:

1. **Base Layer** - Rounded square with brat green (#8ACE00) gradient
2. **Specular Highlight** - White radial gradient overlay (top-left)
3. **Glass Blur** - Translucent blur layer
4. **Inner Glow** - Subtle white border/glow

Import these into Icon Composer and adjust Liquid Glass properties:
- Specular highlights: High
- Blur: Medium
- Translucency: ~85%
- Shadows: Subtle
